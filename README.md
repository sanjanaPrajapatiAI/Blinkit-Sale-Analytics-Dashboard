# Blinkit Sales Analytics Dashboard

## Overview

The Blinkit Sales Analytics Dashboard is an interactive Power BI dashboard developed to analyze Blinkit's sales performance, customer behavior, and business insights. The dashboard provides a comprehensive view of sales trends, product performance, outlet analysis, and customer purchasing patterns through interactive visualizations.

---

## Project Objectives

- Analyze overall sales performance.
- Monitor customer purchasing behavior.
- Identify top-performing products and categories.
- Compare outlet performance.
- Track sales across different locations.
- Provide business insights using interactive dashboards.

---

## Tools & Technologies

- Power BI
- Microsoft Excel / CSV
- Power Query
- DAX
- Data Modeling
- Data Visualization

---

## Dashboard Features

- Total Sales KPI
- Average Sales
- Total Orders
- Average Rating
- Sales by Product Category
- Sales by Outlet Size
- Sales by Outlet Type
- Sales by Outlet Location
- Fat Content Analysis
- Item Type Analysis
- Interactive Filters (Slicers)
- Dynamic Charts and KPIs
